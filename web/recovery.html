Recovery Actions

Assess risk and execute recovery strategies for overdue accounts

Workflow Guide
  1. View Loan Accounts: Check the table below for Account IDs linked to Borrowers
  2. Find Overdue Loans: Look for "Delinquent" status with Days Past Due > 0
  3. Assess Risk: Copy Account ID & Borrower ID → Fill form below → Click "Assess Risk"
  4. Execute Strategy: Based on risk level, select appropriate recovery strategy
  5. View History: Enter Borrower ID to see all communications sent
